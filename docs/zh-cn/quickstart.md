# 快速开始

>
使用无验阻API可以帮助您轻松绕过CAPTCHA，现已支持hCAPTCHA、reCAPTCHA、Turnstile等产品。本文档提供了[HTTP API](/zh-cn/#)
以及~~[SDK](/zh-cn/quickstart)~~的详细使用方法。

### 支持的CAPTCHA版本

+ reCAPTCHA
    + 通用版
    + 企业版
    + 定制版
+ hCAPTCHA
    + 通用版
+ Cloudflare
    + JS质询（Web应用防火墙）
    + Turnstile

### 获取APIKEY

访问[无验阻API控制台](https://console.captchabypass.net/#/)注册账户获取APIKEY。
